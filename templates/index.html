Steps to Create a Simple Web Page with Flask:

Install Flask:

  • Open a terminal or command prompt.
  • Install Flask using the following command: pip install Flask

Create the HTML Template:

  • Create a folder named 'templates' in the same directory as your 'a.py'.
  • Inside the templates folder, create a file named 'index.html'.
  • Open 'index.html' and write a simple HTML structure:
    • <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Webpage</title> </head> <body> <h1>Hello, World!</h1> </body> </html>

    Run your Flask App:

    • In the terminal, run the Flask app with the following command:python a.py
    • Open your web browser and go to http://127.0.0.1:5000/. You should see your simple web page.
